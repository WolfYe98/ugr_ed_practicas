vectorD
Rep del TDA vectorD

iterador final (siguiente al ultimo) sobre elementos no nulos

Funcion de Abstraccion

Un objeto f del TDA vectorD debe cumplir:

-f.n_ele es el número de elementos del vectorD

-f.vd es una lista de pair<int, T>, donde int es la posicion del vector y T es el contenido de dicha posición.

-f.v_nulo es el valor del vectorD que es nulo

Invariante de Representación

Un objeto válido rep del TDA vectorD representa una lista solo con valores distintos de v_nulo y n_ele>=0

f.n_ele debe ser mayor o igual a 0.

f.vd no puede tener elementos que sean distintos de valor nulo. vd->second != v_nulo Se accede mediante: {f.vd->first, f.vd->second}
